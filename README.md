# Sequelize-gigs

Project structrue is as follows:
  - config
    -> database.js
  - models
    -> Gig.js
  - public
    - CSS
      -> style.css
    - img
      -> img.jpg
  - routes
    -> gigs.js
  - views
    - Layout
      -> landing.handlebars
      -> main.handlebars
    - add.handlebars
    - error.handlebars
    - gigs.handlebars
    - index.handlebars
  - app.js
  - package.json

  - express-handlebars: "^3.0.0"
  - handlebars: "^4.7.7"
  - pg: "^8.11.0"
  - pg-hstore: "^2.3.4"
  - sequelize: "^6.32.1"
